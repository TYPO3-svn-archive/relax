Demonstrates how to use CouchDB with your Extension.

Get your free CouchDB for testing purposes on www.couchone.com

This Extension uses libraries from the PHP on Couch Project https://github.com/dready92/PHP-on-Couch

Todos:
* Reading from Couch
* Add localization features
* Add replication features
* Simple Object storage
* Add jQuery dependency
* Add JS Code
* OpenOffice Documentation
